# Exception Management Workflow

## Exception Types
- Reconciliation mismatch
- Duplicate payment
- Invalid transaction
- Compliance breach
- Access violation

## Workflow
1. Detect exception in process or reconciliation.
2. Log exception with type, severity, and related entity.
3. Assign to responsible role with SLA.
4. Investigate and document root cause.
5. Resolve with correction or reversal.
6. Approve resolution and close case.

## Evidence
- Exception record with timestamps.
- Resolution notes and approval.
- Linked audit events.
