# One-Page Architecture Summary

## Purpose
Provide a compact view of the architecture for executives and stakeholders.

## MVP Architecture
- Modular monolith with event-ready boundaries.
- Ledger-centric accounting core and immutable audit trail.
- Rules-driven product engine for savings and loans.
- Compliance evidence captured at transaction time.

## Core Modules
- Member Management and KYC
- Share Capital and Savings
- Loans and Collections
- Cashiering and Treasury
- Accounting and Reporting
- Governance and Compliance

## Key Controls
- Maker-checker for sensitive actions
- Segregation of duties
- No edits to posted transactions
- Period close locking

## Future Extraction Targets
- Notifications
- Document or OCR
- Reporting
- API gateway
