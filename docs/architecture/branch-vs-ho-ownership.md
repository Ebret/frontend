# Branch vs Head Office Data Ownership

## Ownership Matrix

| Data Area | Branch Role | Head Office Role | Notes |
| --- | --- | --- | --- |
| Member onboarding | Membership Officer | Compliance Officer | HO can override approvals with audit trail |
| Savings transactions | Teller | Treasury Officer | HO can set limits and freeze accounts |
| Loan approvals | Credit Committee | General Manager | HO defines approval thresholds |
| Accounting postings | Accounting Officer | CFO or Controller | HO controls period close |
| Compliance reports | Compliance Officer | Board Secretary | HO signs official submissions |

## Control Rules
- Branch can create transactions but cannot alter posted entries.
- Head Office can lock periods and apply global policy changes.
- All overrides require documented justification.
