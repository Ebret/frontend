# Data Migration Playbook

## Preparation
1. Inventory legacy sources and data owners.
2. Define migration waves and cutover window.
3. Agree on reconciliation rules and sign-off process.

## Migration Waves
1. Reference data and chart of accounts.
2. Active members and KYC documents.
3. Savings and share capital balances.
4. Loan balances and schedules.
5. Opening GL balances.

## Validation
- Tie-out subledger totals to GL.
- Validate member identity deduplication.
- Run parallel reporting for one period.

## Cutover
- Freeze legacy updates.
- Execute final delta import.
- Produce signed reconciliation pack.
