# Security and Privacy Controls

## Data Protection
- Encrypt TIN, government IDs, and sensitive PII at rest.
- Mask sensitive fields by default in UI.
- Log access to PII with actor, role, and reason.

## Access Control
- Enforce role-based access and segregation of duties.
- Require MFA for privileged roles.
- Use time-bound sessions and device logging.

## Data Retention
- Retain financial records per regulatory requirements.
- Define retention windows for logs and documents.
- Support legal hold for investigations.

## Incident Response
- Record all security events and anomalies.
- Provide audit trails for access and changes.
